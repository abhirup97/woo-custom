**Woocommerce Custom Plugin**

== Installation ==

== Minimum Requirements ==

* PHP 7.4 or greater is required (PHP 8.0 or greater is recommended)
* MySQL 5.6 or greater, OR MariaDB version 10.1 or greater, is required

== Manual installation ==
Manual installation method requires downloading the [Image Processing] (here is the plugin link) plugin and uploading it to your web server via your favorite FTP application. 

== User Manual ==
admin can add additional fee from woocommerce > settings > general > Custom fee per order
the amount will be shown in checkout page.

Customers can add custom meta data from a single product page and it will be saved and the admin can check out this from the order section.
